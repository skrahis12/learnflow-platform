# LearnFlow Platform

A premium online learning platform designed to help you unlock your potential with world-class instructors.

## Features

- **Interactive Dashboard**: Track your course progress and achievements.
- **Comprehensive Course Catalog**: Browse courses across Development, Design, Data Science, and more.
- **Responsive Design**: Learn on any device, anywhere.
- **Modern Tech Stack**: Built with React, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd learnflow-platform
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Routing**: React Router DOM
